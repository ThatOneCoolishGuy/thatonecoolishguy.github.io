Welcome Home.

Here at The Software of Liberty Foundation, our goal is to provide everyone with free and open source software. Our software is mainly for Linux CLI and Android, and sometimes for Linux desktops. We provide software that gives you these three freedoms: The right to distribute the software in any way, shape, or form, the right to modify the software in any way, and, the right to use the software however you'd ever like to.

How to use:

In order to use our free and open source software, you need a Linux computer or Android mobile device.

Using our Linux apps:

To use our Linux apps, you need to first download the app of your choice. Then you need to give permission to launch the app. To do run the command, "sudo chmod +x [APP_NAME_HERE]". Lastly, run the file by typing the following command, "./[APP_NAME_HERE]". Enjoy your new and open source app!

Using our Android apps:

To use our Android apps, you first need to download the app of your choice, then click on it in Downloads. Make sure to enable apps from unknown sources. Lastly install the app and click "Open."
